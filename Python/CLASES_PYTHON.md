# herencia
---
1. cuando se crea el personaje gerrero da error al crearlo por que no se le atribuyen los atributos, luego de crear otra funcion init y darle los atributos funciona
2. la superclase o la clase padre es la que de ella se originan las hijas o subclases,  esta tiene la informacion base de las cuales se basan las hijas
3. la instruccion "pass" funciona como un paro momentaneo a las funciones, es como decirles que esperen un momento para desarrollarlos mas adelante
4. la funcion "super()" es la que tiene a la superclase o clse padre, esta funcion nos ayuda a facilitar el tener que  especificar las clase padre siendo mas eficiente
5. la herencia multiple hace que una clase hija pueda heredar de mas de una clase a la vez
6. aplicar herencia en POO nos ahorra miucho tiempo y codigo ademas de hacerlo mas reutilizable, eficiente, estructurado y flexible, viniendo bastante bien para la programacion orientada a objetos
