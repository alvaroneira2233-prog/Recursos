# QUE ES UN UML Y PARA QUE SIRVE?
---

lo tomaremos como el ejemplo de unos planos para constuir una casa, tendremos que brindarle la informacion necesaria 
para que el diagrama sea entendible y especificarla ya que tenemos varios tipos de diagramas, y cada uno
cumple una funcion diferente a la hora de querer mostrar informacion, tenemos como ejemplos:
